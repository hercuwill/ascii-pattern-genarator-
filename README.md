# ascii-pattern-genarator-
A Java command-line tool that generates various ASCII patterns using asterisks.

## Features
- Print rectangles (5x7)
- Print downward right triangles
- Print upward right triangles
- Generate random rectangles with varying dimensions
- Interactive menu-driven interface

## How to Run
1. Compile: `javac ascii-pattern-generator.java`
2. Run: `java Project2`
3. Follow the on-screen prompts to select a pattern
